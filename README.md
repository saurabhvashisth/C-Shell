# C Shell
A fully functional Interactive Unix Bash like Shell in C. 
Features: 
1. Signal Handling.
2. Multi-threading.
3. Process creation(forking).
4. Background & Foreground processes.
5. User defined & built-in commands.
6. I/O Redirection.
7. Piping.

HOW TO USE
1. Compile the file bash_shell.c with gcc.
2. Run ./a.out
